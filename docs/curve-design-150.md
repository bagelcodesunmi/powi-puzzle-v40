# 150레벨 구간 설계 가이드 (D7 리텐션 커브)

> 작성: 2026-07-19 (Director 승인). 150레벨을 D7까지의 소비 곡선에 맞춰 4구간으로 나눈 설계 기준.
> 전제: 보드 단조 증가(3→4→5→6, 최대 6×6), min ≤ 8(솔버 검증선), 예고봇 % = 난이도 체감 지표,
> 내 돌 HP 1 고정. 적 개체 속성(hp/전투)·혼합 배치 사용 가능 (`checklist-enemy-traits.md`).

## 구간표

|------|------|----------|------------|-----|------|------|
| **초보** | 1~30 | D1 | 90~100% | 1~4 | 3→4→5 | 규칙 전부 학습 — 실패는 "사고"여야 함 |
| **하수** | 31~70 | D2~D3 | 55~85% | 3~5 | 5×5 | 규칙 조합 응용 — 실패→재도전 루프 시작 |
| **고수** | 71~115 | D4~D5 | 35~60% | 4~6 | 5×5 → **6×6 데뷔(L101)** | 수읽기 — 원거리 벽·순서 퍼즐·함정 회피 |
| **초고수** | 116~150 | D6~D7 | 15~40% (각인판 0%대) | 5~8 | 6×6 정착 | 희생 수순·다중 턴 수읽기 — 성취가 상품 |

## 구간별 설계 원칙

### 초보 (L1~30) — D1
- 현재 L1~L10이 앞 1/3. 새 개념 1개당 "이식 레벨"(익숙한 판 + 새 규칙 — L6~L8 방식) 원칙.
- 전 레벨 "정확히 N턴" (min−1 우호 5,000판 0승) — 요행 클리어 금지, 학습 무결성.
- 벽은 전부 적 인접(초반 문법). 채움 재고: 카탈로그 90%+ 인접벽 442개.

**L11~30 데뷔 맵 (Director 확정, 2026-07-19)**

| 레벨 | 내용 | 대역 |
|------|------|------|
| 11~15 | 전투 응용 소화 (L6~10에서 배운 조합, 톱니 상승) | 90%+ |
| **16** | **적 HP2 데뷔** — 5×5 이식판. "공격수 2로 밀폐 → 연타 즉사"가 자연스러운 유일한 길인 판. 반 박자 늦은 두 번째 화살이 그 자체로 튜토리얼 | 90%+, min 리셋(2~3) |
| 17~18 | HP2 응용 (확산 관리 + 2공격수 계획) | 85%+ |
| **21** | **혼합 속성 데뷔** — 비전투/전투 공존, "안전한 쪽부터" 순서 학습 | 90%+ |
| 30 | 초보 관문 — HP2+전투 종합 | 70~85% |

- HP2 데뷔 근거: ① 볼리·반격 체득(L6~10)이 선행 조건 ② 초보 구간 novelty 박자 균등(16→21→30) ③ D1 안에서 "적도 단단해질 수 있다" 인상 심기.
- **잠금 교훈은 데뷔판에서 가르치지 않는다**: 데뷔판은 성공 경험 전용. "공격수 1로 밀폐 = 잠금"(안티패턴 6-1)의 존재는
  하수 구간에서 잠금 위험이 명백히 보이는 판으로 따로 가르친다 (실패→재도전 학습, 불공평감 없는 배치).

### 하수 (L31~70) — D2~D3 (리텐션 승부처)
- **4레벨 사이클**: 응용 2 + 신선한 변형 1 + 숨돌리기(90%+) 1. 10의 배수 레벨 = 관문(구간 기믹 종합).
- 기믹 응용: HP2+전투 조합, 혼합 배치 순서 퍼즐, 2기 구조 변형, 원거리 벽 첫 등장(읽기 시험 예고편).
- 분산형 허용 시작 (우호 확산 −1턴 요행 OK — 재도전 다양성).

### 고수 (L71~115) — D4~D5
- 이탈 원인은 난이도가 아니라 반복감 — **novelty 밀도를 올린다**.
- 기믹 데뷔: **금 간 벽**(적만 부술 수 있는 1히트 경계벽 — 봉쇄에 타이머, 2026-07-20 엔진 구현 완료),
  겹벗기기 각인판(후보 L 계열), 자충 유도(backlog 검증판), **6×6 첫 등장**(벽 프리셋 3~4개로 min≤8 유지).
  적 3기는 관문식 옵션 (적 수 정책 절).
- 함정의 전환: 차폐 코너·HP2 잠금이 보드에 "존재"하고, 그걸 만들지 않는 것이 퍼즐 (함정 회피 = 수읽기).
- L105~115 = 초고수 진입 심사 구간 (연속 40%대).

### 초고수 (L116~150) — D6~D7
- min 6~8 (검증 가능 상한), 예고봇 0%대 각인판을 5레벨 간격 배치 (사이는 30%대 완충).
- 6×6 + 적 2~3기 + HP2 + 혼합 — 전 기믹 총동원. L150 = 6×6 피날레 보스.
- 시드 풀 6~8개 (재도전이 곧 플레이 루프).

## 보드 크기 정책 — 6×6 데뷔 = L101 (Director 확정, 2026-07-19)

"어려워질수록 보드도 커진다"는 체감은 유지하되, **보드는 150레벨에서 한 번 남은 카드**임을 전제로 운용한다.

- 보드 단조 증가 + 최대 6×6 규칙상 진행은 3→4→5→6 네 단계뿐이고, 현재 커브가 이미 L5에서 5×5 도달
  → L5~L150의 남은 보드 카드는 **5→6 한 번**.
- 난이도의 실체는 보드 크기가 아니라 **예고봇 %(수읽기 깊이)** — 카탈로그 실측상 5×5 안에 100%~3% 전 대역이 존재.
  보드 확대는 난이도 장치가 아니라 **novelty 비트 + min 상한 확장 장치**다.
- **6×6 데뷔 = L101** (고수 중반):
  1. 데뷔판은 L5 방식의 **완충판** — 벽 프리셋 많은 90%급으로, "새 보드"와 "새 어려움"을 동시에 주지 않는다.
  2. 데뷔 이후 전 레벨이 6×6 (되돌림 불가) → 6×6 물량은 50개(L101~150)로 고정 — 검증 비용(벽 프리셋으로
     min≤8 유지 필수) 감당 가능선.
  3. D5 진입 시점(≈L100)의 갱신감과 일치.
- 요약: **보드는 L101에서 마지막으로 한 번 커지고, 어려움은 수읽기 깊이로 계속 오른다.**

## 턴 수(예산) 정책 — 톱니형 상승 (Director 확정, 2026-07-19)

구간이 오를수록 턴 수(min)는 커진다 — 단 아래 세 원칙과 함께.

- **톱니형 상승**: 구간 안에서는 계단으로 오르되, **새 기믹 데뷔에서는 턴 수를 리셋**한다
  (현행 커브의 실례: 1·2·3·4·3·**1**·2·3·4·5 — 전투 데뷔 L6에서 1로 리셋).
  "새 규칙 + 긴 판" 동시 부여 금지 — 학습이 깨진다. 구간 min 대역: 초보 1~4 / 하수 3~5 / 고수 4~6 / 초고수 5~8.
- **턴 수 ≠ 난이도**: 같은 min 4에 예고봇 100%(후보 M)와 2.3%가 공존한다 (카탈로그 실측).
  턴 수가 주는 것은 판의 깊이감·실수 기회·성취감이고, 실제 어려움은 예고봇 %가 조절한다.
  후반 구간 = "min 크게 + 예고봇 낮게"의 조합 (길고 깊은 판).
- **min은 결과값** (v43 가이드 §13): 배치를 먼저 만들고 솔버 min이 목표 대역에 들어오는지 확인한다.
  숫자를 먼저 정하고 배치를 끼워 맞추지 않는다. 상한은 검증선 때문에 **8 고정**.

## 적 수 정책 — 기본 상한 2기, 3기는 관문식 옵션 (Director 확정, 2026-07-19 레드팀/블루팀 검토)

- **기본 상한 2기** (2기부터 벽·코너 구조 필수 — 열린 판 2기는 발산 검증됨).
- **3기는 6×6(L101+) 한정 옵션 카드**. 아래 3중 관문을 전부 통과한 판만 채용:
  ① 솔버 min≤8 확정 ② 파티클 랜덤 분리(✅ 완료 2026-07-20) 기반 시드 검증 ③ 실플레이 판읽기 부하 확인.
- 초고수 설계 시점에 관문 통과 판이 안 나오면 **자동으로 2기 상한 확정** (폴백 — 커브는 hp2·혼합·수읽기로 성립).
- 레드팀 요지(기록): 3기는 "나쁘다"가 아니라 "아직 증명 안 됨" — 검증 리스크(발산·상태공간), §14-2 시드 한계(다중 포획),
  제작 비용(카탈로그 없음), 모바일 판읽기 부하. 블루팀 요지: 초고수 스펙터클 카드, 보드 정책 정합, 6×6 밀도 여유.

## 벽 정책 — 하드 상한 4개, 용도별 대역제 (Director 확정, 2026-07-19)

| 용도 | 벽 수 | 근거 |
|------|------|------|
| **일반 채움판** (커브 대부분) | **0~2개** | 카탈로그 전수 검증 범위와 일치 — 후보 검색·검증 자동화 대역. 90%+ 학습판도 인접 벽 1~2에 집중 (실측) |
| **밀폐 튜토리얼** (1탭 데뷔판) | 3개 | 3면 밀폐 = L1·L6 문법. 이 용도 외 3개 사용 금지 |
| **금 간 벽** (고수+) | ≤3개 (일반 벽 상한과 별도) | 위상 상태 2³ — 검증선 유지. 겉모습(호박색 균열)으로 구분 |
| **6×6 (L101+)** | **3~4개** | min≤8 검증선 유지용 필수 보정 — 6×6은 벽 없이 솔버 확정 불가 |
| **하드 상한** | **4개** | 5개부터 "미리 풀린 그림" + 시각 노이즈로 읽기 힌트 기능 상실 |

- 상한 4는 미적 취향이 아니라 **제작 파이프라인 경계선**: 벽 0~2 = 양산 대역(카탈로그 자동화),
  3~4 = 특수 목적 수제 대역(개별 검증 필수, 허용 용도는 밀폐 튜토리얼·6×6 보정 두 가지뿐).
- 벽 위치 원칙은 별도 유지: 인접 벽 = 초반 문법, 원거리 벽 = 중후반 읽기 시험 (`catalog-guide.md` §4).

## 시드 정책 — 판의 성격으로 나눈다 (Director 확정, 2026-07-19 레드팀/블루팀 검토)

**판 유형 이원화** — 고정 퍼즐(저작)과 변주 퍼즐(채움)의 장점을 분리 수취:

| 판 유형 | 시드 | 밸런스 도구 | 턴 수 |
|---------|------|------------|------|
| **저작판** — 기믹 데뷔·관문·각인·보스 (구간당 3~5개, 전체의 ~20%) | **1개 + mirror: false** | solveWithSeed + 실플레이 (그 퍼즐 하나를 정밀 튜닝) | "정확히 N턴" 문자 그대로 |
| **채움판** — 커브의 나머지 ~80% | 구간 연동 풀: 초보 2 / 하수 3 / 고수 4~5 / 초고수 6~8 | 적대 min + 예고봇 대역 + 시드 전수 PASS (분포 밸런스) | 분산형 허용 |

- **번호 체계**: base = 레벨번호 × 100,000, +1부터 순차 탐색해 검증 PASS만 채용 (L16 → 1600001~).
- **풀 큐레이션**: 채움판 시드는 PASS 중에서 고른다 — 요행성 과다 시드는 배제 가능 (풀 내 편차 통제).
- **런타임 규칙** (현행 명문화): 실패 탭 재시도 = 같은 시드 / 클리어 후 재방문 = 미사용 시드 우선 / 풀 소진 시 전체 재사용.
- **선행 과제**: 파티클 랜덤 분리 — 중간 포획 판(2기·HP2·혼합)의 시드 인증이 실플레이 없이 성립하려면 필수.
  ✅ **완료(2026-07-20)**: 연출 전용 `fxRng` 분리, 로직 `prng`는 미러링·확산 전용. L11~20 제작 준비됨.
- 레드팀/블루팀 요지: 시드 1 전면화는 수읽기→암기 변질·완충 밸브 소멸·예고봇 % 체계 붕괴,
  풀 전면화는 검증 비용·저작 통제 희석 — 판 성격 분리로 양쪽 공격 모두 회피.

## 반복·다양성 정책 — 지루함 방지 (Director 착안, 2026-07-20)

이식 레벨(익숙한 판 + 새 규칙)은 학습에 필수지만, **같은 위치·구조·턴수가 연속되면 첫 플레이에서 지루함**을 준다.
학습용 반복은 살리되 지루함이 시작되기 전에 끊는다.

- **구조 반복 3연속 금지**: 같은 적 위치(미러 정규화 기준)나 같은 판 구조가 **3레벨 연속** 등장하지 않게 한다.
  2연속까지는 "재발견/이식"으로 허용, 3연속부터는 반드시 변주 축을 바꾼다.
- **반복 판은 미러 방향을 서로 다르게 고정** (Director 착안): 연속되는 이식/재사용 판은 기본 미러 방향을
  일부러 다르게 둬서 **첫 플레이 실루엣이 매번 다른 사분면**에 찍히게 한다.
  (주의: `mirror: true`는 재도전 다양성만 주고, 첫 진입 방향은 시드가 정하므로 첫 플레이 반복은 못 가린다.
  저작판·연속 반복 구간은 `mirror: false` + 방향을 직접 다르게 지정하는 편이 확실하다.)
- **의도된 재사용은 감정 반전과 함께**: 같은 판을 다시 쓸 땐 감정 태그를 바꿔 "같은 판인데 이번엔 다르다"를
  만든다 (예: L5 neutral → L8 dopamine "이번엔 반격한다"). 감정이 안 갈리는 반복이 진짜 지루함이다.
- **변주 축 우선순위**: 보드·예산은 단조 증가 제약이 있으므로, novelty는 ① 적 사분면 로테이션 ② 벽 프리셋 형태
  ③ 미러 방향 ④ 기믹(HP2·혼합·금 간 벽) ⑤ 예고봇 % 대역 순으로 만든다.

**측정 도구**: `emotion-curve.html`의 "🔁 반복 경보" 섹션이 적 위치(미러 정규화)·판 구조·**구조 이식**
(보드 크기만 다른 같은 로컬 패턴 — L3→L5식 완충 이식을 잡는 크로스 보드 비교)·감정 3연속을 점수화한다.
완화(-1)는 **반복 상대와** 감정이 반전됐을 때만 적용된다("같은 판인데 이번엔 다르다"가 성립할 때).
새 레벨 추가 시 **경보 점수 2 이상(주의)·4 이상(높음)** 구간을 확인하고 변주를 넣는다.
현재 커브 실측 (2026-07-21 L5·L8 복도 판 교체 후): 주의는 L8 하나 — L5 재사용이지만 감정 반전(neutral→dopamine)으로

## 설계 근거 (고정 원칙)

1. **검증선 일치**: 전 구간 min ≤ 8 → 150레벨 전부 적대 솔버 확정 가능 ("검증된 레벨만 출시" 유지).
2. **난이도 축 역할 분담**: 초보→하수는 *배치 복잡도*(벽·적 수), 고수→초고수는 *수읽기 깊이*(예고봇 % 하강).
   min은 8을 넘지 않는다 — 판은 길어지지 않고 **깊어진다**.
3. **재고 현실성** (카탈로그 실측): 90%+ 442개 / 중간대 1,809개 / 심화 1,420개.
   부족한 것은 채움이 아니라 **기믹 데뷔판** (구간당 3~5개, 수제 설계 + 전체 검증 파이프라인 통과 필수).
