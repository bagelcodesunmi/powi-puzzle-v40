> 대상: `powi-puzzle-v42.html`, `level-lab-v3.html`

> 작성일: 2026-07-17

## 왜 v42를 따로 만들었나

`v41`은 기존 확정 흐름을 유지하는 메인 빌드다.

`v42`는 초반 1~3레벨의 학습 품질을 높이기 위한 실험 버전이며, 특히 아래를 검증하기 위해 분리했다.

- 1레벨을 *즉시 성공형*으로 더 명확하게 만들 수 있는가
- 벽을 *장애물*이 아니라 *튜토리얼 가이드*로 사용할 수 있는가
- 1→2→3레벨의 학습 난이도 상승을 더 부드럽게 만들 수 있는가

즉 `v42`는 메인 빌드 대체가 아니라, 초반 튜토리얼 재설계안의 실험 브랜치다.

---

## 초반 1~3레벨 설계 목표

### 레벨 1
- 플레이어가 마지막 1칸을 두면 즉시 포획이 발동하는 구조
- 실패보다 성공 경험을 우선한다
- 규칙을 설명하기보다, *한 번 눌러서 바로 이해*하게 만든다

### 레벨 2
- “아무 데나 두는 것”이 아니라 “닫히는 칸을 찾는 것”을 배우게 한다
- 벽은 답을 숨기는 장치가 아니라, 막아야 할 방향을 줄여주는 장치다

### 레벨 3
- 보드를 4×4로 확장하되 벽은 힌트 수준만 제공한다
- 플레이어가 직접 구조를 완성하는 첫 레벨 역할을 맡는다

---

## v42 초반 튜토리얼 원칙

### 1. 벽은 초반에 "제약"보다 "가이드"다
초반 1~3레벨에서 벽은 플레이어를 괴롭히는 기믹이 아니라,
다음 중 하나의 역할만 가져야 한다.

- 남은 1칸을 눈에 띄게 만든다
- 막아야 할 통로를 줄여준다
- 포위 외곽선이 어떻게 생기는지 암시한다

### 2. 1레벨은 성공 보장형 도파민 샘플이다
- 첫 입력 후 바로 반응이 나와야 한다
- 플레이어가 “내가 뭘 한 거지?”보다는 “아 이렇게 되는구나”를 느끼게 해야 한다

### 3. 2레벨은 오답 학습이 가능해야 한다
- 틀릴 수는 있지만 왜 틀렸는지가 바로 보여야 한다
- 함정처럼 느껴지면 안 된다
- 정답은 1개여도, 정답을 읽는 근거는 눈에 보여야 한다

### 4. 3레벨부터 자립을 시작한다
- 벽이 거의 다 해주면 안 된다
- 다만 완전 자유 퍼즐로 급전환하면 안 된다
- "벽이 힌트를 줬고, 나머지는 내가 만들었다" 느낌이 나야 한다

## 레벨별 현재 초안

### L1 — 마지막 1칸 포획
```js
{ board: 3, enemies: [[0,0]], walls: [[1,0,1],[1,1,1]], schedule: [1], spread: true, combat: false, min: 1, exactBudget: true }
```

### L2 — 닫는 칸 읽기
```js
{ board: 3, enemies: [[0,0]], walls: [[1,0,1]], schedule: [2,1], spread: true, combat: false, min: 3, exactBudget: true }
```

### L3 — 4x4 진입 + 벽 힌트
```js
{ board: 4, enemies: [[1,2]], walls: [[0,2,1]], schedule: [2,2,2], spread: true, combat: false, min: 6, exactBudget: true }
```

## level-lab-v3에서 확인해야 할 체크리스트

### L1 체크
- 남은 한 칸이 시각적으로 읽히는가
- 너무 자동처럼 느껴지지 않는가
- 첫 탭 후 즉시 발동/쾌감이 오는가

### L2 체크
- 정답 칸을 읽을 근거가 보이는가
- 오답 칸이 있어도 억울하지 않은가
- 확산 전에 통로 봉인 개념이 자연스럽게 느껴지는가

### L3 체크
- 4×4 확장 부담이 과하지 않은가
- 벽 도움을 받되 플레이어가 직접 푼 느낌이 있는가
- 2레벨에서 3레벨로 갈 때 점프가 과하지 않은가

## 안티패턴

다음은 v42 초반 재설계에서 피해야 한다.

### 1. 벽이 규칙 예외처럼 보이는 경우
플레이어가 벽을 “포위 구조의 일부”로 이해하지 못하면 튜토리얼이 실패한다.

### 2. 2레벨이 함정 문제처럼 보이는 경우
정답이 1개인 것과, 정답이 불친절한 것은 다르다.

### 3. 3레벨이 갑자기 자유 퍼즐이 되는 경우
1~2레벨이 가이드형이었는데 3레벨이 갑자기 완전 자립형이 되면 학습 흐름이 끊긴다.

### 4. 벽에 공격 제한 등 새 규칙을 너무 많이 실는 경우
초반 1~3레벨에서는 벽을 우선 공간 가이드로만 사용한다.
공격 제한/고급 상호작용은 뒤로 미룬다.
